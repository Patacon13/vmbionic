# VM con Ubuntu Bionic e interfaz gráfica

Pasos para ejecutar:

1. Clonar repositorio.
```
git clone https://github.com/Patacon13/vmbionic
```
2. Levantar vm.
```
vagrant up
```
3. Esperar que la terminal termine de trabajar.
4. Iniciar sesión
```
Usuario: vagrant
Contraseña: vagrant
```
5. Iniciar interfaz
```
startx
```
